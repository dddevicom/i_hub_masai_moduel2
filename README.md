# Data Preprocessing Assignment

## Overview
Cleaned, encoded, and scaled a customer dataset using pandas and sklearn.

## Tasks
- **Task 1:** Data Cleaning — handled missing values, removed duplicates
- **Task 2:** Encoding — One-Hot Encoding on `city`, Label Encoding on `gender`
- **Task 3:** Scaling — MinMaxScaler and StandardScaler on `age` and `annual_income`

## Libraries Used
- pandas
- scikit-learn

## How to Run
Open the `.ipynb` file in Jupyter Notebook and run all cells sequentially.
